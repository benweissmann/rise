EasyRubygame by benweissmann

A video game framework built around Rubygame, designed for educational use and beginner programmers.

EasyRubygame (ERG) is a framework built on top of the Rubygame library, which is based on Pygame. ERG is designed to take care of the repetitious tasks of game building and allow developers to focus on the logic. ERG is designed for new game developers and programming students. It was created to assist with an introductory programming class at Concord Academy in Concord, MA. Some teaching materials may be available for educators; contact the author if interested.

Dependencies

Rubygame (installable via rubygems)

Install

Download the latest source. For each rubygame project, copy the "src" folder from the rubygame tarball. Change the files in the the "src" directory, then run "runner.rb" to start your game!
